# 🪐 Star Wars Planet Wiki

Una wiki sobre planetas de Star Wars. Funcionalidad CRUD basica que te permite manejar planetas,
y guardarlas en tu dispositivo


## 📱 ¿Qué hace la app?
Permite tener una enciclopedia de planetas en tu móvil:
* **Listar y Buscar:** Ves todos los planetas y películas guardados, y puedes buscarlos por nombre usando la barra de búsqueda superior.
* **Crear:** Puedes añadir nuevos planetas (con foto, clima, terreno, etc.) y películas.
* **Editar:** Si te has equivocado, puedes cambiar los datos.
* **Borrar:** Para eliminar entidades con pulsación larga.
* **Navegación Global:** Tiene un menú lateral para cambiar rápidamente entre Inicio, Planetas, Películas, Ajustes y Sobre Nosotros.
* **Notificaciones:** La app te avisará con una notificación del sistema cuando crees una entidad correctamente.
* **Validación de Integridad:** Si intentas crear un planeta o película con el mismo nombre de uno ya existente, la app te avisará con un diálogo para evitar duplicados.

## 🛠️ Tecnologías que he usado
Estoy aprendiendo a usar lo último de Android:
* **Kotlin** (El lenguaje principal).
* **Jetpack Compose** (Para la interfaz).
* **Room** (Para guardar los datos en el móvil.Una persistencia real).
* **Hilt** (Para la inyección de dependencias, algo complicada de entender pero muy util).
* **Corrutinas y Flow** (Para el manejo de datos asíncronos).
* **Material Design 3** (Diseño).
