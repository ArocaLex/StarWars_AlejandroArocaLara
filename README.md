# 🪐 Star Wars Planet Wiki

Una wiki sobre planetas de Star Wars. Funcionalidad CRUD basica que te permite manejar planetas,
y guardarlas en tu dispositivo


## 📱 ¿Qué hace la app?
Permite tener una enciclopedia de planetas en tu móvil:
* **Listar:** Ves todos los planetas guardados.
* **Crear:** Puedes añadir nuevos planetas (con foto, clima, terreno, etc.).
* **Editar:** Si te has equivocado, puedes cambiar los datos.
* **Borrar:** Para eliminar planetas.
* **Navegación:** Tiene un menú lateral y una barra superior que cambia según la pantalla.

## 🛠️ Tecnologías que he usado
Estoy aprendiendo a usar lo último de Android:
* **Kotlin** (El lenguaje principal).
* **Jetpack Compose** (Para la interfaz).
* **Room** (Para guardar los datos en el móvil.Una persistencia real).
* **Hilt** (Para la inyección de dependencias, algo complicada de entender pero muy util).
* **Corrutinas y Flow** (Para el manejo de datos asíncronos).
* **Material Design 3** (Diseño).
